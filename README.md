# Handwritten Digit Recognition — MNIST

A feedforward neural network (MLP) built with PyTorch to classify handwritten digits from the MNIST dataset.

## Result
- **Test Accuracy: 98.04%**
- Trained for 20 epochs using Adam optimizer

## Model Architecture
- Input → 784 (flattened 28x28 image)
- FC1 → 284 neurons (ReLU)
- FC2 → 128 neurons (ReLU)
- FC3 → 10 neurons (output)

## Tech Stack
- Python
- PyTorch
- Torchvision
